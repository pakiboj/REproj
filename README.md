# REproj
SRB:

Projekat za predmet Računarska elektronika

U projektu realizovan je:
    BinToHex konvertor u asemblerskom koodu za x86 arhitekturu.
    Visuelni prikaz tog konvertora.

Program je napisan koristeći Irvine32.inc biblioteku u MASM ekstenziji za Visual Studio 2017.


ENG:

Project for the course Computer Electronics

The project implements:

    BinToHex converter written in assembly code for the x86 architecture.
    Visual representation of that converter.

The program was written using the Irvine32.inc library in the MASM extension for Visual Studio 2017.
